Method of execution:
Run build.xml file, then run PacmanGame.jar. Documentation is in build/docs/index.html

How to play:
After executing game, to play firstly choose file with the map you want to play on. Then press button "Start game". 
You will be playing as pacman, to move use Arrows. For some additional help click anywhere on gamefield with left mouse button.
